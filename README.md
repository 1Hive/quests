## Setup & Launch

1. Clone

   > git clone https://github.com/1Hive/quests.git

2. Install

   > yarn install

3. a. Launch with **Rinkeby** test network

   > yarn start

   b. Launch with local chain

   > yarn start:local
   >
   > [new terminal]
   >
   > yarn chain
   >
   > [new terminal]
   >
   > yarn deploy
