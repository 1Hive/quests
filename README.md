Quests is a bounty board for 1Hive swarms.

## Setup & Launch

Clone the repository

```sh
git clone git@github.com:1hive/quests
```

Install the dependencies

```sh
yarn
```

Start the project

```sh
# Rinkeby testnet
yarn start
# Local chain
yarn start:local &; yarn chain &; yarn deploy &
```
